# 🎮 Jogo da Velha

Este projeto é uma implementação do **clássico Jogo da Velha**, desenvolvida utilizando **HTML, CSS e JavaScript**. A aplicação permite que dois jogadores disputem em um tabuleiro 3x3, alternando entre os símbolos **X** e **O** até que um deles vença ou ocorra empate.

O objetivo do projeto é praticar conceitos básicos de **desenvolvimento web** e **manipulação do DOM com JavaScript**.

---

## 📌 Funcionalidades

* Tabuleiro interativo 3x3
* Alternância automática entre os jogadores
* Verificação de vitória
* Detecção de empate
* Botão para reiniciar a partida
* Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout
* **JavaScript** – Lógica e funcionamento do jogo


3. Abra o arquivo **index.html** em seu navegador.

---

## 🎯 Como jogar

1. O jogo começa com o **jogador X**.
2. Clique em qualquer espaço vazio do tabuleiro para fazer sua jogada.
3. Os jogadores se alternam automaticamente entre **X** e **O**.
4. O vencedor é aquele que conseguir alinhar **três símbolos na horizontal, vertical ou diagonal**.
5. Caso todas as posições sejam preenchidas sem vencedor, a partida termina em **empate**.

---

## 🔄 Reiniciar o jogo

Para iniciar uma nova partida, basta clicar no botão **"Reiniciar"**, que limpa o tabuleiro e reinicia o jogo.
